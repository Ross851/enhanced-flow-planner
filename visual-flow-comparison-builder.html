💡 KEY LEARNING
ALWAYS filter at source!
• Use OData filters in Get Items: $filter=CustomerID eq 'ABC123'
• Use $top=1 to get only first match
• Use $select to get only needed columns
• Use Filter Array for in-memory filtering (no API calls)
• NEVER loop through all items to find one!
🚨 THE MULTIPLICATION RULE
Apply to Each = MULTIPLICATION, not addition!
• 1000 items × 1 action = 1,000 API calls
• 1000 items × 3 actions = 3,000 API calls
• Nested: 100 × 50 × 2 actions = 10,000 API calls!

Every action inside Apply to Each happens FOR EVERY ITEM!
